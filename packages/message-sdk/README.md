# @openclaw/message-sdk

OpenClaw application message SDK.

Built on top of `@openclaw/pair-sdk`.

## Includes

- OpenClaw chat message type
- OpenClaw chat payload helpers
- OpenClaw chat app module for the shared dispatcher

This package is where app/business protocol should evolve without changing the generic pairing transport core.

## Build

```bash
npm --prefix packages/message-sdk run build
```

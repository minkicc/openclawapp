# Security Policy

## Supported Versions

We currently provide security updates for:

| Version | Supported |
| --- | --- |
| main branch | Yes |
| latest release | Yes |
| older releases | No |

## Reporting a Vulnerability

Please report vulnerabilities privately by email:
`ruiqiangzh@gmail.com`

Include:

- Affected version
- Reproduction steps
- Potential impact
- Suggested fix (if any)

We will acknowledge reports within 5 business days and provide status updates
as investigation progresses.

Please do not disclose security issues publicly until a fix is available.
